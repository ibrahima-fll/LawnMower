# LawnMower
This project is a simple implementation of a lawnmower management.

## Usage
Run the following commands:
* `sbt assembly`
* `java -jar target/scala-2.12/LawnMower-assembly-0.1.jar "-f" "path-to-lawnmower-config-file"`
